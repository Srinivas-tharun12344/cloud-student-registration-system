## 🚀 Project Overview
A full-stack cloud application to register and manage student data, deployed on AWS.

## 🛠 Tech Stack
- Node.js
- Express.js
- AWS EC2
- AWS DynamoDB
- HTML, CSS, JavaScript

## ✨ Features
- Student Registration
- View Students
- Delete Students
- Cloud Deployment
- REST API Integration

## 🌐 Live Demo
🔗 http://54.208.140.54:3000

## 🏗️ Architecture
Frontend → Node.js (Express) → AWS EC2 → DynamoDB

## 📂 Project Structure
- server.js → Backend API
- index.html → Frontend UI
- DynamoDB → Database

## ⚙️ Setup Instructions
```bash
npm install
node server.js
```

## 🚀 Key Highlights
- Deployed full-stack application on AWS EC2
- Integrated DynamoDB for scalable storage
- Implemented REST APIs for CRUD operations
